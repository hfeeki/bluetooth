Class: Bluetooth

Bluetooth

new Bluetooth()

Bluetooth class includes all useful bluetooth global interfaces.

Please note that the application should not create Bluetooth object, BC manages the object model.

Properties:
Name Type Description
devices Array.<Device> The advertising devices, this is filled after 'BC.Blueooth.StartScan' called
