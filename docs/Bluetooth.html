Class: Bluetooth

Bluetooth

new Bluetooth()

Bluetooth class includes all useful bluetooth global interfaces.

Please note that the application should not create Bluetooth object, BC manages the object model.

Properties:
Name Type Description
devices Array.<Device> The advertising devices, this is filled after 'BC.Blueooth.StartScan' called
services Array.<Service> The services add by 'AddService' interface
isopen boolean Bluetooth is open or not

<static> StartScan(type, uuids)

Starts a scan for Bluetooth LE OR Classical(only in android platform) devices.
Parameters:
Name Type Argument Description
type array <optional>
 Is LE scan or classical scan, please input "LE" OR "Classical", if you want use default scan strategy, just ignore this param.
uuids array <optional>
 Array of services to look for. If null or [], it will scan all devices(just support BLE service UUID)
Source:
  • bc.js, line 794
Fires:
  • Bluetooth#event:newdevice
Example
BC.Bluetooth.StartScan();//start hybrid scan,in android platform the default scan strategy is scan for BLE device 5s first,
then scan for classical device 12s,and restart the hybrid scan untill the BC.Bluetooth.StopScan be called. in IOS platform only start the LE scan.
BC.Bluetooth.StartScan(null,["00000000-0000-0000-0000-000000000000"]);//start hybrid scan with LE service UUID.
BC.Bluetooth.StartScan("LE"); //just start the LE scan
BC.Bluetooth.StartScan("Classical"); //just start the classical scan in android platform, if the platform is IOS,call this method will nothing happen.

	}
