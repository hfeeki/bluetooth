Class: Device

Device

new Device(deviceName, deviceID, advertisementData)

Device represents the remote BLE Peripheral device.

Please note that the application should not create Device object, BC manages the object model.

Parameters:
Name Type Description
deviceName string The name of the device
deviceID string The ID of the device(ID is assigned by the smart phone,if there is no ID, it is recommended to new the device instance after obtaining devices' information from BC.Bluetooth.startScan)
advertisementData object The device advertisement data, includes LocalName, TxPowerLevel, IsConnectable, ServiceData, ManufacturerData, ServiceUUIDs, SolicitedServiceUUIDs, OverflowServiceUUIDs
Properties:
Name Type Description
deviceName string The name of this device
deviceID string The ID of this device
services Array.<Service> The services of this device
isConnected boolean If this device is connected
isPrepared boolean If this device is prepared ('prepared' means this device object can be used to access the services' characteristics)
connectSuccessCallback function This success callback function will be called after this device is well prepared
systemID DataValue The system ID of this device
modelNum DataValue The model number of this device
serialNum DataValue The serial number of this device
firmwareRevision DataValue The firmware revision of this device
hardwareRevision DataValue The hardware revision of this device
softwareRevision DataValue The software revision of this device
manufacturerName DataValue The manufacturer name of this device
